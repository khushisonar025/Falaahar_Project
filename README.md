Falaahar Project
Overview:
Falaahar is an online platform designed to help rural areas purchase fresh fruits and vegetables conveniently.
The website ensures secure transactions through email and OTP verification. It includes three dedicated panels for efficient management: Admin, Delivery, and User Interface.

Key Features:
🛒 E-commerce Functionality – Allows users to browse, select, and purchase fresh fruits and vegetables online.
🔐 Secure Authentication – Implements email verification and OTP-based login for enhanced security.
🏢 Three-Panel Architecture:
Admin Panel – Manages products, orders, users, and delivery assignments.
Delivery Panel – Allows delivery personnel to track and update order status.
User Interface – Enables customers to browse products, place orders, and track deliveries.
📩 Order Confirmation via Email – Sends order details and updates to users via email using PHPMailer.
🚚 Delivery Management System – Assigns deliveries efficiently with a limit on active deliveries per delivery person.
📊 Dashboard for Admin & Delivery Personnel – Displays order statistics, pending deliveries, and user details.
💰 Discounts & Offers Section – Displays ongoing offers to attract customers.
🖥️ User-Friendly UI – Built with PHP, HTML, CSS, Bootstrap, and JavaScript for a seamless experience.
🛠️ Database Management – Uses MySQL to store user details, orders, and product information efficiently.
🌍 Localized Approach – Focuses on small cities and rural areas to make fresh produce accessible.
